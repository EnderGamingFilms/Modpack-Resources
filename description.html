Setup & Installation

CURSEFORGE LAUNCHER:  The Twitch Launcher is being replaced with curseforge's own launcher which you can download here:  https://download.curseforge.com/

This is the easiest way to install Wallend 3.  Link your Minecraft game, go to browse modpacks, find Wallend 3 and install, click play, that's it!

Anyone moving from the Twitch launcher to the CurseForge launcher will still automatically keep all profiles and modpacks.
